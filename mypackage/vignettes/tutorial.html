Introduction

This is the first part of the Project 3, where it includes tutorials for three functions that is in the mypackage. They are: my_t.test, my_lm, and my_knn_cv.

Tutorial for my_t.test

A tutorial for my_t.test testing the hypothesis that the mean body_mass_g of Adelie penguins is equal to 4000 (where the alternative is that the mean is less than 4000). Carefully interpret the results using a p-value cut-off of $$= 0.05.

Our null hypothesis is that the mean body_mass_g of Adelie penguins is equal to 4000 And our alternative is that the mean is less than 4000.

# Hypothesis teasing using my_t.test
p_value <- my_t.test( x = (na.omit(my_penguins[species = Adelie])$body_mass_g), "less", 4000)$p_val

Since we have p-value = 1, which is greater than \(\alpha = 0.05\). Hence, the result is not statistically significant and we failed to reject null hypothesis.

Tutorial for my_lm

A tutorial for my_lm using flipper_length_mm as the independent variable and body_mass_g as the dependent variable.

# Regression Model Using flipper_length_mm as independent variable and 
# body_mass_g as dependent variable.
lm_model <- my_lm(body_mass_g~flipper_length_mm, data = my_penguins)
lm_coefficients <- lm_model[,1]
lm_model
#>                      Estimate  Std.Error   t.value      Pr...t..
#> (Intercept)       -5780.83136 304.918998 -18.95858  2.716317e-55
#> flipper_length_mm    49.68557   1.513958  32.81833 1.050292e-107

Then, we will be doing a two-sided hypothesis testing to see the relationship between flipper_length_mm and body_mass_g. Our null hypothesis is that flipper_length_mm has no significant influence on body_mass_g. And our alternative hypothesis is that flipper_length_mm does have significant influence on body_mass_g.

From the table, we can know that the coefficient for flipper_length_mm is 49.6855664. We can interpret that coefficient as the expected difference in the response between two observations differing by one unit in flipper_length_mm where other covariates stays the same.

Also, since the Pr(>|t|) value for flipper_length_mm comes from the two-sided t test. And it’s value of 0 is smaller than the \(\alpha = 0.05\). Hence, we can reject the null hypothesis and in favor of our alternative hypothesis. Thus, we would say there’s correlation between flipper_length_mm and body_mass_g

Tutorial for my_knn_cv

A tutorial for my_knn_cv using my_penguins. Predict output class species using covariates bill_length_mm, bill_depth_mm, flipper_length_mm, and body_mass_g.

#load data and clean it
data <- my_penguins[, c("species", "bill_length_mm", "bill_depth_mm", "flipper_length_mm", "body_mass_g")]


# clear all the NAs
data <- na.omit(data)
train = data[,2:5]
cl = data$species
train_error <- rep(NA, 10)
cv_error <- rep(NA, 10)
n <- nrow(data)
for (i in 1:10) {
  my_knn <- my_knn_cv(train = train, cl = cl, k_nn = i, k_cv = 5)
  cv_error[i] <- my_knn$cv_err
  error = 0
  for (j in 1:n){
    if (my_knn$class[j] != cl[j]) {
      error = error+1
    }
  }
  train_error[i] = error/n
}

# Record our error in a table
table_error <- cbind(c(1:10), train_error, cv_error)
table_error
